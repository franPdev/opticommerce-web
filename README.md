# 🚀 OptiCommerce AI

**SaaS B2B de Gestión y Optimización Inteligente de Stock para Retail**

OptiCommerce AI es una plataforma de software orientada al sector retail y e-commerce que soluciona de forma definitiva el problema de la gestión ineficiente del inventario. Utilizando Agentes de Inteligencia Artificial, la plataforma analiza datos históricos, previene quiebres de stock, genera estrategias de liquidación automáticas y predice futuras compras para maximizar el ROI.

---

## ✨ Características Principales

- 📊 **Dashboard de Alta Conversión**: KPIs gerenciales en tiempo real (Valor Total, Riesgo de Stock, Proyección a 30 días) con un diseño *Dark Mode* premium e intuitivo.
- 🤖 **Predicciones con IA**: Análisis de demanda que recomienda con exactitud qué, cuánto y cuándo comprar mediante órdenes de compra sugeridas.
- ⚠️ **Alertas Inteligentes de Stock**: Sistema de notificaciones en tiempo real para productos próximos a caducar, sobrestock o en riesgo inminente de quiebre (Stockout).
- 🚀 **Estrategias de Liquidación Autoejecutables**: La IA detecta inventario estancado y genera campañas de marketing automatizadas (emails, redes sociales) que pueden ejecutarse con un solo clic.

## 🛠️ Stack Tecnológico

Este proyecto fue construido utilizando tecnologías de vanguardia enfocadas en la escalabilidad, la robustez tipada y el alto rendimiento:

- **Frontend Core**: React 19 + TypeScript + Vite.
- **Styling & UI**: Tailwind CSS v4 (Design System customizado con CSS-in-JS para máxima flexibilidad) y Lucide Icons.
- **Data Visualization**: Recharts para gráficos dinámicos y proyecciones.
- **AI & Logic**: Integración simulada nativa preparada para **Gemini 1.5 Pro** y Webhooks (n8n/Make).
- **Arquitectura**: Clean Architecture y estado global gestionado mediante Context API.

## 🚀 Instalación y Ejecución Local

Si deseas correr este proyecto de forma local en tu máquina:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/opticommerce.git
   cd opticommerce
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Visualizar la aplicación:**
   Abre tu navegador y navega a `http://localhost:5173`. Las credenciales de demostración (`admin@optistock.ai` / `demo1234`) se cargarán automáticamente en la pantalla de inicio de sesión.
